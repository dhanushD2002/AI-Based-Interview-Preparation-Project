# AI-Based Interview Preparation Project

## 📌 Project Overview
This is a full-stack AI-Based Interview Preparation System developed to help users practice interview questions and improve technical skills.

The application consists of a frontend and backend, connected using REST APIs.

---

## 🛠 Tech Stack

### Frontend
- Angular
- HTML
- CSS
- TypeScript

### Backend
- Java
- Spring Boot
- REST APIs

### Database
- MySQL

---

## 📁 Project Structure

ai-interview-prep  
│  
├── ai-interview-prep-frontend   (Angular frontend)  
└── ai-interview-prep-backend    (Spring Boot backend)

---

## ✨ Features

- User Registration & Login
- Interview Question Generation
- REST API Integration
- Frontend–Backend Connectivity
- MySQL Database Support

---

## 🚀 How to Run

### Backend

1. Open backend folder
2. Configure database in application.properties
3. Run Spring Boot application:

```bash
mvn spring-boot:run


Frontend

Open frontend folder

Install dependencies:

npm install


Start Angular app:

ng serve


Open browser:

http://localhost:4200

🎯 Purpose

This project was created for learning full-stack development and interview preparation, focusing on REST APIs, Angular frontend, Spring Boot backend, and MySQL integration.
